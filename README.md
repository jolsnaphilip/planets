# Solar System Planets Table

This project is a simple HTML and CSS webpage that displays information about the planets in our solar system in a table format.

## Features

* Displays details of planets such as category, diameter, temperature, and number of moons.
* Styled using CSS for a clean and attractive design.
* Hover effects and modern table styling.

## Technologies Used

* HTML
* CSS

## Purpose

I created this project to practice HTML tables and improve my CSS styling skills by designing a better-looking webpage.

## How to Run

1. Download the project files.
2. Open the `index.html` file in a web browser.
3. View the planets table.
